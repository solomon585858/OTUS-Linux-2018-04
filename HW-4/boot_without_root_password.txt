Способы попасть в систему без пароля:

1. Загрузка в Emergency mode(target).
- При появлении меню GRUB2 при загрузке надо нажать клавишу e.
- Добавить параметр systemd.unit=emergency.target в конце строки linux16.
- Нажать Ctrl+x для загрузки с указанным параметром.

2. Загрузка в Rescue mode(target) (первый способ).
- При первом способе при загрузки системы надо сразу выбрать пункт меню для загрузки в rescue mode.

3. Загрузка в Rescue mode(target) (второй способ).
- При втором способе при появлении меню GRUB2 при загрузке надо нажать клавишу e.
- Добавить параметр systemd.unit=rescue.target в конце строки linux16.
- Нажать Ctrl+x для загрузки с указанным параметром.

4. Загрузка в Debug Shell.
- При появлении меню GRUB2 при загрузке надо нажать клавишу e.
- Добавить параметр systemd.debug-shell в конце строки linux16.
- Нажать Ctrl+x для загрузки с указанным параметром.

5. Смена пароля root (если забыли) с помощью rd.break.
- При появлении меню GRUB2 при загрузке надо нажать клавишу e.
- Удалить параметры rhgb и quiet в строке linux16.
- Добавить параметр rd.break enforcing=0 в конце строки linux16.
- Нажать Ctrl+x для загрузки с указанным параметром.
- Смонтировать файловую систему как writeable с помощью команды mount -o remount,rw /sysroot.
- Сменить root файловой системы с помощью команды chroot /sysroot.
- Сменить пароль с помощью команды passwd.
- Ввести команду touch /.autorelabel.
- Смонтировать файловую систему как read-only с помощью команды mount -o remount,ro /.
- Дважды нажать команду exit.
- Загрузиться в систему в обычном режиме и ввести новый пароль для root.
